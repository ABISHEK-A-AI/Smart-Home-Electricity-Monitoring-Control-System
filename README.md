# Smart Home Electricity Monitoring & Control System

## Project Overview

This project is an ESP32-based Smart Home Electricity Monitoring and Control System.

The system monitors:

- Voltage
- Current
- Power Consumption
- Energy Units

The system also allows remote control of appliances through Arduino IoT Cloud.

## Components Used

- ESP32
- Voltage Sensor
- Current Sensor
- 16x2 LCD Display
- 2 Channel Relay Module
- WiFi
- Arduino IoT Cloud

## Features

- Real-time voltage monitoring
- Real-time current monitoring
- Power calculation
- Energy unit calculation
- Electricity cost estimation
- Remote appliance control
- LCD display monitoring
- EEPROM data storage

## Working

1. Sensors measure voltage and current.
2. ESP32 calculates power and energy usage.
3. Values are displayed on the LCD.
4. Data is sent to Arduino IoT Cloud.
5. Users can control appliances remotely using the cloud dashboard.
